Clase de redes neuronales 12/09/2024

la clase de hoy aprendi sobre redes neuronales primero : Importación de bibliotecas y carga de datos Se importan las bibliotecas necesarias, como TensorFlow y Keras, para trabajar con redes neuronales. Luego, se carga el conjunto de datos CIFAR-10, que consta de 60,000 imágenes de 32x32 píxeles en 10 clases diferentes Preprocesamiento de datos Las imágenes se normalizan y se dividen en conjuntos de entrenamiento y prueba. Las etiquetas se convierten a formato categórico Se define una arquitectura de red neuronal convolucional utilizando Keras Sequential API. La red incluye capas convolucionales, de agrupación máxima y densas, con activaciones ReLU y una capa de salida con activación softmax para clasificación multiclase El modelo se compila con un optimizador Adam, una función de pérdida categórica cruzada y métricas de precisión. Luego, se entrena el modelo utilizando el conjunto de datos de entrenamiento durante 10 épocas El modelo entrenado se evalúa utilizando el conjunto de datos de prueba, obteniendo métricas de pérdida y precisión Finalmente, se muestra cómo utilizar el modelo entrenado para predecir las clases de nuevas imágenes
